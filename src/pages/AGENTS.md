# src/pages/AGENTS.md

## Scope

These instructions apply to Astro pages and route files inside `src/pages`.

## Page edit controls

Keep page work local.

Default budget:

- Read at most 2 page files.
- Edit at most 1 page file.
- Do not follow imports unless you explain why.
- Do not alter routing, slugs, redirects, or dynamic route behavior unless asked.
- Do not change SEO metadata unless the request is about SEO or page positioning.
- Do not run a build for copy or layout edits.

## Homepage narrative order

The preferred homepage flow is:

1. Brand-first governed AI hero
2. Why brand-first governance matters
3. Brand AI Readiness Assessment
4. Role-Intent Testing
5. iBOM framework
6. Start with brand and extend governance by function
7. Latest Thinking
8. Final CTA

## CTA hierarchy

Primary CTA:

```txt
Assess your brand's AI readiness
```

Secondary CTA:

```txt
Explore iBOM
```

Use utility CTAs like `Get in touch` only in contact, footer, and final contact zones.

Avoid:

```txt
Begin the journey
```
