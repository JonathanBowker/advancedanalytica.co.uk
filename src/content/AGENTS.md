# src/content/AGENTS.md

## Scope

These instructions apply to content collection files, Markdown, MDX, and structured content inside `src/content`.

## Content edit controls

Do not bulk edit content unless explicitly asked.

Default budget:

- Edit at most 1 content file.
- Preserve slugs.
- Preserve frontmatter schema.
- Do not change `publishedAt`, `draft`, categories, or tags unless asked.
- Do not add invented data, citations, clients, or statistics.
- If a source is needed, add a TODO note instead of inventing one.
